== Description ==

RestroPress - Clover allows you to import and real time export of data between restropress and clover pos . Here are some of the options available with this plugin.

* Set  your merchant ID and API token.
* Import customer details, food item details, category, and order details from clover pos by using respective button.
* Export real time customer details, food item details, category, and order details from restropress to clover pos

== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Click on upload and then choose the downloaded file and click on upload.
3. Click activate.
